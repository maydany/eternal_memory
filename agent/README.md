# Agent Cognitive Layer

이 디렉토리는 에이전트의 "뇌"입니다. 프로그래밍적으로 사용되는 파일들이 저장됩니다.

## ⚠️ 중요 사항

- **자동 관리**: 이 파일들은 에이전트가 자동으로 관리합니다.
- **개인정보**: Git에서 자동으로 제외됩니다 (.gitignore).
- **백업 권장**: 중요한 프로필 정보가 포함되어 있으므로 정기적으로 백업하세요.

## 📁 파일 설명

### USER.md (현재)
- **목적**: 에이전트의 사용자 이해 모델 (Theory of Mind)
- **내용**: 장기적으로 가치 있는 사용자 선호도, 작업 패턴, 기술 컨텍스트
- **업데이트**: 자동 (매일 새벽 3시 consolidation 시)
- **사용**: 모든 LLM 호출 시 시스템 프롬프트에 주입

**Quality Threshold:**
- Confidence >= 0.7 (70% 이상 확신)
- Evidence >= 3 (최소 3번 이상 관찰)

### SOUL.md (향후 확장)
- **목적**: 에이전트의 성격과 행동 규칙
- **업데이트**: 반자동 (사용자 승인 필요)

### MEMORY.md (향후 확장)
- **목적**: 절대 잊어서는 안 되는 핵심 사실들
- **업데이트**: 수동 + 자동

## 🔄 데이터 흐름

```
대화 → Buffer → flush_buffer()
                      ↓
          timeline/YYYY-MM-DD.md (모든 대화)
                      ↓
          (매일 새벽 3시 consolidate)
                      ↓
              LLM 분석 + Quality Check
                      ↓
                 USER.md (선별된 인사이트만)
```

## 🛡️ 보안

- 파일 권한: 700 (소유자만 읽기/쓰기)
- 외부 스크립트 실행 시 주의
- 백업은 암호화된 공간에 저장 권장

## 📊 vs Human Layer

| | Agent Layer (여기) | Human Layer (user_memory/markdown) |
|---|---|---|
| **목적** | 프로그래밍 소스 | 시각화/노트 |
| **사용** | 코드에서 읽음 | 사람이 읽음 |
| **내용** | 큐레이션됨 | 전체 로그 |
| **업데이트** | 자동 | 자동 미러링 |
